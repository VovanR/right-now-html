# right-now-html

> I need it yesterday

## Install

```
git clone git://github.com/vovanr/right-now-html project-name
cd project-name
rm -rf .git
git init
npm install
```

## Usage

- Markup in `index.jade`
- Styles in `style.styl`

Run
```
hs .
open index.html
```

## Build

```
npm run build
```
